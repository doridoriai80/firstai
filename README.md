# Rule + LLM 기반 AI Chatbot

## 소개
이 프로젝트는 간단한 규칙 기반 응답과 LLM API(OpenAI GPT)를 결합한 AI 챗봇입니다.

## 구성
- 규칙 기반 응답 (`rule_engine.py`)
- OpenAI API를 통한 LLM 연동 (`llm_api.py`)
- 메인 실행 (`main.py`)

## 실행 방법
1. `config.py`에 OpenAI API 키 입력
2. `python main.py` 실행

## 향후 개선 아이디어
- GUI 인터페이스 추가
- 사용자 입력 로그 저장
- 챗봇 성능 평가
